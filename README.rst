aiohttp toolbox
===============

|BuildStatus| |Coverage| |pypi|

Tools for aiohttp I want to reuse.

Installation
------------

Requires **python 3.6** or **python 3.7**.

.. code:: shell

    pip install aiohttp-toolbox

To test::

   atoolbox --root tests/demo web
